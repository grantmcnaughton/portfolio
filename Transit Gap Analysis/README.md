### Transit Gap Analysis Using ArcGIS

Using ArcGIS and data from the US Census American Community Survey, this geospatial analysis of transit coverage gaps across the Research Triangle region of North Carolina synthesizes multiple data layers into actionable planning recommendations. By integrating transit route networks, commuter travel time distributions, and walkability metrics, a composite spatial model identifying underserved communities and prioritizing corridors for potential service expansion was developed.
 
This project utilizes public raster and vector datasets, reflecting my ability to independently scope and execute a data-driven study, interpret spatial relationships in a real-world policy context, and communicate technical findings clearly, including an honest assessment of the analysis's limitations and the factors a more comprehensive study would need to address.
